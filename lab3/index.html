Lab 3 - File Structures and File Transfer

Challenge

The challenge of this lab is to create a local file structure on our computers and add index.html files.

Problems

Creating the local file structure was easy. However, formatting the images of the HTML source code and websites to fit and be readable on this website took some time. We managed to solve this issue by limiting the width of the images to a certain amount.

Reflection

Overall, this assignment was pretty easy aside from getting all of the pictures of the HTML source code and websites and formatting them. It was straightforward, given that we had a template for the file structure and the basic CSS for our webpages.
